👥 Personel Kayıt ve Yönetim Sistemi
Python ve PySide6 (Qt) kullanılarak geliştirilmiş, personel verilerini CSV formatında depolayan ve yöneten bir masaüstü uygulamasıdır.

🛠️ Teknik Özellikler
Dil: Python

Arayüz: PySide6 (Qt for Python)

Veri Saklama: CSV tabanlı yerel veritabanı yapısı

Validasyon: TC Kimlik ve Telefon numarası gibi alanlar için özel doğrulama kuralları (QRegularExpressionValidator).
